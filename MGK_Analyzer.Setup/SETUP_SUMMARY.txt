ISSUE: "WiX Toolset not found"
SOLUTION:
  1. Download from: https://wixtoolset.org/releases/v3.14/
  2. Install WiX314.exe
  3. Verify: dir "C:\Program Files (x86)\WiX Toolset v3.14\bin"

ISSUE: "Cannot find MGK_Analyzer.exe"
SOLUTION:
  1. Build main project first:
     dotnet build MGK_Analyzer.csproj -c Release
  2. Try building setup again: build.bat

ISSUE: "Access Denied" during build
SOLUTION:
  1. Run as Administrator
  2. Check file permissions
  3. Close any running instances of the app
